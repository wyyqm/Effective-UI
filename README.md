# effectiveUI

## 作用

对 `element-ui 2.x` 进行扩展, 新增如下组件:

-   ButtoUnion: 常用按钮合集
-   InputUnion: 常用输入框合集
-   SelectUnion: 常用下拉框合集
-   DatePickerUnion: 常用日期选择器合集

yarn install
